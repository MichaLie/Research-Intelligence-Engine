## How to Use This Index

1. **Find a session**: Look up by topic or date
2. **Check status**: See if investigation is complete
3. **Navigate**: Go to `sessions/[session-name]/journal.md` for full details
4. **Resume**: Use "Continue [topic]" command to resume any active/paused session
